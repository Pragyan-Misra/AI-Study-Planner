# AI-Study-Planner
AI-powered study planner that predicts learning style using ML and generates personalized study plans and timetables using rule-based logic.
🎓 **Personalized Study Recommendation System + Daily Timetable Generator**

This project is an AI-powered system designed to enhance student productivity by generating personalized study plans and daily timetables. It combines **Machine Learning** and **rule-based logic** to understand user behavior and recommend the most effective study strategy.

The system predicts a student's **learning style** using a trained Random Forest model based on inputs such as study hours, attendance, motivation, and stress levels. It then generates a customized study plan and an optimized timetable tailored to the user’s preferences.

Additional features such as **attention span, preferred study time (morning/evening/night), and focus subject prioritization** are incorporated using rule-based logic to make the system more practical and user-centric.

✨ **Key Features:**

* 🧠 Learning style prediction using Machine Learning
* 📚 Personalized study plan generation
* ⏰ Automated daily timetable creation
* 🎯 Focus subject prioritization
* 🧠 Attention span-based study sessions
* 🌅 Time preference-based scheduling
* 🌐 Interactive UI built with Streamlit

🛠 **Tech Stack:**

* Python
* Pandas, NumPy
* Scikit-learn (Random Forest)
* Streamlit (Frontend UI)
* Pickle (Model persistence)

🚀 This project demonstrates a hybrid approach by combining **data-driven predictions** with **rule-based decision-making**, making it both intelligent and adaptable to real-world student needs.
